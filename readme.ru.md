# 🧠 MarkovTextGenerator — N-грамная языковая модель

Легковесная вероятностная языковая модель, реализованная на чистом Python без использования тяжелых NLP-фреймворков. Генерация текста базируется на статистическом анализе N-грамм и марковских процессах.

## 🧠 Математическая суть
Модель рассчитывает условную вероятность появления следующего слова на основе фиксированного окна предшествующего контекста:

$$P(w_i | w_{i-(n-1)}, \dots, w_{i-1})$$

### Особенности архитектуры
* **Инвариантность к размеру контекста ($N$):** Класс `NGramLanguageModel` параметризован. Легко переключается между режимами биграмм ($N=2$), триграмм ($N=3$) и глубокого контекста.
* **Разделение фаз fit/generate:** Стадия обучения (`.fit()`) выполняется единожды, генерация происходит за константное время $O(1)$ с использованием хэш-таблиц.
* **Отказоустойчивость:** При попадании в «тупиковое состояние» модель не завершает работу, а осуществляет случайное блуждание (мягкую интерполяцию) для перехода к валидному контексту.

## 📈 Сравнение контекста
* **Биграммы ($N=2$):** Контекст 1 слово. Высокая энтропия, текст получается хаотичным.
* **Триграммы ($N=3$):** Контекст 2 слова. Повышается локальная связность, сохраняются фразеологизмы и синтаксис оригинала.

## 🚀 Как запустить
Проект не требует установки сторонних библиотек.
1. Положите текстовый файл для обучения в `text.txt`.
2. Запустите скрипт:
```bash
python markov_generator.py
```
3. Результат генерации запишется в файл generated_text.txt
